    - FizzBuzz問題
        - 以下のルールに従って、1から100までの数字を順番に出力するプログラムを作成してください。

        - 5の倍数のときは数字の代わりに「Buzz」と出力
        - 3と5の両方の倍数（すなわち15の倍数）のときは「FizzBuzz」と出力
        - それ以外の数字はそのまま出力
